# FlowBoard PM release notes

## 0.1.1

- Added board-level project configuration.
- Project chips now use configured project colors.
- Board title and description can be edited from the top toolbar.

## 0.1.0

Initial release.

- Linear-style project board.
- Local `.pmboard` files.
- Backlog, Todo, In Progress, and Done lanes.
- Card project, owner, stage, type, priority, due date, and tags.
- Drag-and-drop status updates.
- Card search, create, edit, and delete.
